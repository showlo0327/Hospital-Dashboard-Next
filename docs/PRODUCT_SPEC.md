# PRODUCT_SPEC

## Layout (Implemented)

- Header: full width, fixed, 64px, glassmorphism
- Sidebar: below header, 260px expanded / 72px collapsed
- Main content: auto-fills remaining width, scrollable
- Inspector: fixed right panel, 316px, xl+ only

## Dashboard Statistics (Complete)

- 5 stat cards: Total Devices (128), Online (121), Offline (2), Warning (5), Average Latency (14ms)
- Glass appearance, hover lift 150ms
- Responsive: 2-col mobile, 5-col xl

## Network Workspace Header (Complete)

- Three-zone layout: Left (title + subtitle), Center (search placeholder), Right (action buttons)
- WorkspaceTitle: page label + title heading
- WorkspaceSubtitle: descriptive meta text
- WorkspaceSearch: placeholder for future global device search (Name, IP, Vendor, Model, Building, Floor, Location, Type, SN, Remarks)
- WorkspaceActionGroup: container for Refresh, Filter, ViewSwitch (List/Grid), More
- WorkspaceViewSwitch: segmented control, exclusive list/grid toggle
- Reusable by all future workspace pages (SNMP, Topology, etc.)
